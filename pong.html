Pong Controls

Spacebar: Start the game

Mouse: to move paddle when game is not paused

Arrow Keys: Move paddle when game IS paused

P: pause

C: change color of the game

Extensible Controls

The sliders at the bottom change the properties of the game dynamically

The first one changes how many pixels per frame the paddle updates towards the ball. Scales from 1 to 10

The second changes how many frames until the paddle skips it's movement on that frame. Scales from 1 to 25

The third changes how the inflection of the ball is changed depending on where it hits a paddle. Scales from 1 to 10 in 0.1 increments

The fourth can advance and rewind time when the game is paused. 0 is the beginning of the game. When the game is paused time can be advanced 500 steps from the current position. The game will continue to record everything as long as it keeps going.
