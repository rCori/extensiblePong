Use the spacebar to serve the ball

Use the mouse or arrow keys to move paddle up and down

The p key pauses the game

Press the c key to change the color of the game

The sliders at the bottom change the properties of the ball dynamically

The first one changes how many pixels per frame the paddle updates towards the ball

The second changes how many frames until the paddle skips it's movement on that frame
